## 六、数据安全架构

### 6.1 五层安全模型

```
Layer 1: 物理安全 (BitLocker加密)
Layer 2: 文件系统安全 (NTFS权限控制)
Layer 3: 数据加密 (AES-256-GCM)
Layer 4: 访问控制 (RBAC + 审计日志)
Layer 5: 传输安全 (TLS/SSL强制)
```
