## 🎨 品牌视觉规范 (Brand Guidelines)

### 配色方案

```
主色调 (Primary):
├── Deep Ocean Blue: #1a365d (专业/信任)
├── Electric Cyan: #06b6d4 (科技/创新)
└── Pure White: #ffffff (简洁/清晰)

辅助色 (Secondary):
├── Success Green: #10b981 (增长/达成)
├── Alert Amber: #f59e0b (注意/提醒)
└── Danger Red: #ef4444 (紧急/重要)

渐变 (Gradients):
├── Primary Gradient: linear-gradient(135deg, #1a365d, #06b6d4)
└── Success Gradient: linear-gradient(135deg, #059669, #10b981)
```

### 字体规范

```
中文:
├── 标题: 思源黑体 Bold (Source Han Sans)
├── 正文: 思源宋体 Regular (Source Han Serif)
└── 强调: 站酷快乐体 (ZCOOL KuaiLe - 用于轻松场合)

英文:
├── 标题: Montserrat Bold / Inter Bold
├── 正文: Inter Regular / Open Sans
└── 代码/数据: JetBrains Mono / Fira Code
```

### 图标/插画风格

```
推荐风格:
✅ 扁平化设计 (Flat Design) + 微渐变
✅ 线性图标 (Line Icons) + 圆角矩形
✅ 等距视角 (Isometric) 用于系统架构图
✅ 数据可视化: 简洁的信息图表 (Infographic Style)

禁止:
❌ 过于卡通/幼稚的风格
❌ 过于复杂的3D渲染
❌ 与科技/AI无关的装饰元素
```

## 📊 A/B测试框架 (Testing Matrix)

### 测试维度

| 维度 | 版本A | 版本B | 版本C |
|------|-------|-------|-------|
| **标题长度** | 短 (<50字符) | 中 (50-80字符) | 长 (>80字符, 含Emoji) |
| **情感基调** | 专业严肃 | 友好亲切 | 紧迫驱动 |
| **CTA按钮** | "了解更多" | "立即体验" | "免费演示" |
| **视觉元素** | 纯文字 | +数据图表 | +真人照片 |
| **社会证明** | 无 | 客户数量("已服务500+") | 具体案例("XX公司增长340%") |
